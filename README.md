# Movies App:
**Android Native** demo mobile application to view and categorize latest and top movies, with the ability to work offline using caching mechanisms.
# Key Technologies
- Kotlin
- Compose UI
- Co-routines
- Hilt
- MVI
- Clean Architecture
- Offline Caching for Data and Images
- Room DB
